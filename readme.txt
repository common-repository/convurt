Use ConvurtGPT to help write your blog posts. Generate content using OpenAI's GPT-3 model directly in the WordPress block editor.

== Description ==

Convurt is a WordPress plugin that allows you to generate content for your blog posts using OpenAI's GPT-3 model. With Convurt, you can easily extend your blog posts with AI-generated content that is contextually relevant and coherent.

Key Features:
- Seamless integration with the WordPress block editor.
- Generate content using OpenAI's GPT-3 model.
- Trigger content generation by typing "/convurt" in the block editor.
- Customize the generated content to fit your needs.
- Content takes your previous blog post content as context to seamlessly integrate with your writing style and flow of your blog post.

== Installation ==

1. Upload the `convurt` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to the Convurt settings page and enter your OpenAI API key.
4. Start using Convurt in the block editor by typing "/convurt" to trigger content generation.

== Usage ==

1. Open the block editor and start writing your blog post.
2. When you want to generate content, simply type "/convurt" and hit enter.
3. Convurt will use the previous content of the blog post as context and generate additional content using OpenAI's GPT-3 model.
4. Review and edit the generated content as needed.

== Frequently Asked Questions ==

= Do I need an OpenAI API key to use Convurt? =

Yes, you need an OpenAI API key to use Convurt. You can obtain an API key by signing up for an account on the OpenAI website.

= Can I customize the generated content? =

Yes, you can edit the generated content directly in the block editor to ensure it aligns with your writing style and the message you want to convey.
